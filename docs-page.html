What is PHP ?

PHP (Hypertext Preprocessor) is a server-side scripting language designed specifically for web development. It is widely used to build dynamic websites and web applications by embedding scripts within HTML pages.

  • PHP code is executed on the server.
  • The result is sent to the browser as plain HTML.
  • It can interact with databases, handle forms, manage sessions, and more.

Output:

 Hello, World!

Why Choose PHP for Web Development ?

Here are some reasons why PHP remains a popular choice:

  1. ✅ Open Source: Free to use with a large community.
  2. ✅ Cross-Platform: Runs on Windows, Linux, macOS.
  3. ✅ Easy to Learn: Simple syntax for beginners.
  4. ✅ Integrated with HTML: Can be embedded directly.
  5. ✅ Database Friendly: Works well with MySQL, PostgreSQL, etc.
  6. ✅ Framework Support: Supports Laravel, Symfony, CodeIgniter, and more.
  7. ✅ Rich Library Support: Built-in and third-party libraries for almost every need.

How PHP Works – The Lifecycle

  1. A user requests a .php page (e.g., index.php) via browser.
  2. The server (e.g., Apache, Nginx) passes the request to the PHP engine.
  3. PHP processes the script on the server.
  4. The result (usually HTML) is sent to the client’s browser.

Note: The client never sees the actual PHP code – only the final HTML.

PHP Basics & Syntax

This section covers the fundamental concepts of PHP syntax, data types, operators, and other key PHP basics.

1. PHP Syntax Overview

PHP code is embedded in an HTML file between <?php and ?> tags. This tells the server to interpret the code as PHP.

Key Points:

  • PHP code must be inside <?php ... ?> tags.
  • PHP statements end with a semicolon (;).
  • PHP is case-sensitive, so $name and $Name are different variables.

2. Comments in PHP

PHP supports two types of comments:

  • Single-line comment using // or #
  • Multi-line comment using /* ... */
Why Use Comments?
  • Documentation: Explains what the code is doing.
  • Temporarily disabling code: Useful for debugging.

3. Variables in PHP

A variable in PHP starts with the dollar sign $ followed by the variable name. Variables are dynamically typed, meaning their type is determined at runtime.

Rules:
  • Variables must start with a letter or underscore ($name, $user_name).
  • Variable names are case-sensitive ($name is different from $Name).

4. Data Types in PHP

PHP supports several data types, including:

  • String: A sequence of characters, enclosed in quotes (" " or ' ').
  • Integer: A whole number (e.g., 42).
  • Float: A number with a decimal point (e.g., 3.14).
  • Boolean: Represents true or false.
  • Array: An ordered collection of values.
  • Object: An instance of a class.
  • NULL: Represents no value.
  • Resource: A reference to an external resource (e.g., a file or database connection).

5. Constants

Constants are identifiers for simple values that cannot be changed during the execution of a script. They are defined using the define() function.

Key Points
  • Constants do not start with a $ symbol.
  • Constants are case-sensitive by default (e.g., SITE_NAME is different from site_name).

6. Strings in PHP

Strings are sequences of characters. They can be defined using single or double quotes.

Single Quotes ('):
  • Does not parse variables inside the string.
  • Escaping special characters is required for certain characters. 
					
$name = 'Alice';
$text = 'Hello, $name'; // Output will be: Hello, $name
Double Quotes ("):
  • Parses variables inside the string.
  • Allows escape sequences like \n (new line) or \t (tab). 
					
$name = "Alice";
$text = "Hello, $name";  // Output will be: Hello, Alice

10. Functions in PHP

Functions are blocks of reusable code that can be called with a specific name.

Defining a Function:
						
function greet($name) {
	return "Hello, $name!";
}
							
echo greet("Alice");  // Output: Hello, Alice!
Default Parameter Values:
						
function greet($name = "Guest") {
	return "Hello, $name!";
}
							
echo greet();  // Output: Hello, Guest!
